# Dealership Marketplace Helper

Load this folder as an unpacked Chrome extension.

Flow:
1. Open `chrome://extensions`
2. Turn on Developer Mode
3. Click `Load unpacked`
4. Select this `facebook-marketplace-extension` folder
5. In the popup, enter the API base and the inventory URL
6. Click `Generate Draft`
7. Open Facebook Marketplace create listing page
8. Click `Apply Draft`

This MVP fills title, price, and description only. It does not auto-submit the listing.
